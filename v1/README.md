# Personal Resume v1

## Giới thiệu

Đây là bản resume cá nhân của mình, được viết bằng framework Streamlit để nhanh chóng tạo ra một 
trang web với giao diện đầy đủ thông tin. Các phiên bản sau sẽ nâng cấp hơn về mặt hình thức.

## Cách sử dụng

1. Cài đặt các thư viện cần thiết:

    ```bash
    pip install -r requirements.txt
    ```

2. Chạy ứng dụng:

    ```bash
    streamlit run app.py
    ```
